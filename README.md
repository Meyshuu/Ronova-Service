# Ronova-Service
=======
# Demo Website Joki Neverness to Everness

Website dasar untuk portal joki dengan:
- homepage berisi menu card layanan
- detail layanan dengan form pesanan, chat customer, dan simulasi pembayaran
- login pengguna dan admin
- dashboard pengguna (saldo, status jokian)
- admin panel (biodata joki, riwayat pekerjaan, update status)
- penyimpanan lokal realtime antar tab melalui `localStorage` dan `BroadcastChannel`

## Cara menjalankan

Python sudah cukup:

```bash
python -m http.server 8000
```

Kemudian buka `http://localhost:8000`.

## Catatan

- Saat ini menggunakan data lokal (demo) gratis.
- Untuk produksi, dapat diganti dengan Firebase, Supabase, atau backend pribadi.
